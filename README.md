# 💪🏻SSAFYGYM 알고리즘 스터디💪🏻
#### 🎯 "코딩 테스트에 합격하셨습니다."</br> 코테 합격을 위한 알고리즘 스터디입니다.
## 📝 스터디 운영
#### 1. 일주일에 5개 문제 풀기: 풀고 난 뒤, 양식에 맞춰 깃허브 업로드
#### 2. 매주 월요일 8시 온라인 미팅 진행: 랜덤으로 알고리즘 선정하여 본인의 풀이 방법 공유
#### 3. 문제 선정: 온라인 미팅 시, 마지막에 알고리즘 유형 선정과 5개 문제 선정
#### 4. 업로드 방식 
  - 폴더명: 
     - 최상위 폴더명은 "N월M주차"
     - 하위 폴더는 각 문제 이름별로 만들기
   - 파일명: 
     - BJ_문제번호_문제이름_본인이름.확장자</br>

     #### EX) 9월1주차/코테합격/BJ_7777_코테합격_김싸피.java
 </br>
